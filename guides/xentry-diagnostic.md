# Mercedes Xentry Diagnostic System

Mercedes Xentry is the official diagnostic platform used for diagnosing, coding and troubleshooting Mercedes-Benz vehicles.

Professional technicians use Xentry to access fault codes, perform programming and view live vehicle data.

## Requirements

• Diagnostic laptop
• Xentry software installed
• Compatible interface or multiplexer
